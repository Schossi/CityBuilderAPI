CityBuilderCore.DistributionComponent Class Reference

Detailed Description

building component that uses SaleWalkers to check what IItemRecipients need
it then gets them from IItemGivers using PurchaseWalkers
finally the SaleWalkers distributes the items to IItemRecipient

https://citybuilder.softleitner.com/manual/resources
