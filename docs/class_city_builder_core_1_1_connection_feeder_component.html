CityBuilderCore.ConnectionFeederComponent Class Reference

Detailed Description

component that feeds into a connection at the buildings points with its value scaled to the buildings efficiency
used on the power station and water pump in the urban demo

https://citybuilder.softleitner.com/manual/connections
